# Adding custom fractions, rather than using auto-detection


## Add fractions to the `pom.xml`

```
<dependency>
   <groupId>org.wildfly.swarm</groupId>
   <artifactId>jaxrs-jaxb</artifactId>
</dependency>
<dependency>
   <groupId>org.wildfly.swarm</groupId>
   <artifactId>cdi</artifactId>
</dependency>
<dependency>
   <groupId>org.wildfly.swarm</groupId>
   <artifactId>jpa</artifactId>
</dependency>
```
